***Intelligent Diagnosis System for Neonatal Auricular Deformities***

**System Overview**

Powered by deep learning technology, this system is designed to assist in the diagnosis of neonatal auricular deformities. It utilizes a binary classification model to determine whether an ear is normal or abnormal, and a six-class classification model to further categorize abnormal ears. Additionally, it provides a deformity severity score (ranging from 0 to 100, where a higher score indicates greater severity) along with heatmap visualization capabilities.

**System Requirements**

    **Hardware Requirements**
        Minimum: Intel Core i5 / AMD Ryzen 3, 8GB RAM, no dedicated GPU required
        Recommended: Intel Core i7 / AMD Ryzen 5, 16GB RAM, NVIDIA GTX 1060 or higher
        Storage: At least 5GB of available space

    **Software Environment**
        Python 3.8 or higher

    **Operating System**
        Windows 10/11, macOS, Linux

**Installation Guide**

    **Install Python**
       If Python is not installed on your computer, please download and install it from the official website: https://www.python.org/downloads/
       During installation, ensure you check the box for "Add Python to PATH".

    **Install Dependencies**
       Open Command Prompt (cmd) or Terminal, navigate to the application's directory, and run:
        Bash
        pip install -r requirements.txt
